# Blog Post de Ejemplo

¡Bienvenidos a mi blog! En este post de ejemplo, exploraremos las capacidades de Markdown eXtended (MDX) para crear contenido increíble. 

## Encabezados

Los encabezados en MDX nos permiten organizar nuestro contenido de manera jerárquica. Podemos utilizar desde el nivel 1 hasta el nivel 6 de encabezados, como se muestra a continuación:

# Encabezado de nivel 1
## Encabezado de nivel 2
### Encabezado de nivel 3
#### Encabezado de nivel 4
##### Encabezado de nivel 5
###### Encabezado de nivel 6

## Texto enriquecido

Podemos aplicar formato al texto para resaltar palabras o hacer énfasis. A continuación, algunos ejemplos:

- *Cursiva*: utiliza un asterisco (*) o un guion bajo (_) alrededor del texto para aplicar formato de cursiva.
- **Negrita**: utiliza dos asteriscos (**) o dos guiones bajos (__) alrededor del texto para aplicar formato de negrita.
- ***Cursiva y negrita***: combina los dos formatos utilizando tres asteriscos (***) o tres guiones bajos (___) alrededor del texto.

## Listas

Podemos crear listas ordenadas y no ordenadas para organizar nuestro contenido. Aquí hay ejemplos de ambos tipos:

### Lista ordenada

1. Primer elemento
2. Segundo elemento
3. Tercer elemento

### Lista no ordenada

- Elemento A
- Elemento B
- Elemento C
